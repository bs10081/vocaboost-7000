## 🎮 使用說明

### ⌨️ 鍵盤操控（桌面）
- **←** 不會（答錯）
- **→** 會（答對）
- **↑** 上一題
- **Space** 翻面
- **G** 朗讀單字

### 📱 觸控手勢（手機/平板）
- **左滑** 不會
- **右滑** 會
- **上滑** 上一題
- **點擊卡片** 翻面

### 📄 功能頁面
1. **首頁** - 查看今日統計和選擇學習模式
2. **開始複習** - 複習今日到期的單字
3. **學習新單字** - 選擇 Level 1-6
4. **收藏難詞** - 複習標記的單字
5. **學習統計** - 查看進度和成就

## 🛠️ 開發指令

```bash
# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 📱 PWA 安裝

### Android (Chrome)
1. 點擊網址列的「⋮」選單
2. 選擇「加到主畫面」

### iOS (Safari)
1. 點擊分享按鈕
2. 選擇「加入主畫面」

**注意**：需先生成 PWA icons（參考 `PWA_ICONS_README.md`）

## 🐛 疑難排解

### 重新啟動開發伺服器
```bash
pkill -f vite
rm -rf node_modules/.vite
npm run dev
```

### 清除瀏覽器快取
在瀏覽器開發者工具中：
1. 按 F12 打開 DevTools
2. 右鍵點擊重新整理按鈕
3. 選擇「清除快取並強制重新整理」
